## 🧠 Методология «Нарратив → Действия → Результаты»

| Компонент | Вопрос | Описание |
|-----------|--------|---------|
| **Нарратив** | *«Зачем я просыпаюсь утром?»* | Определяет ценности и цели, задаёт направление |
| **Действия** | *«Почему я поступаю именно так?»* | Реализация нарратива через ежедневный выбор |
| **Результаты** | *«Это вообще моя жизнь?»* | То, во что превращается нарратив |

**Выбор всегда за вами:** можно изменить свой нарратив, а можно оставить всё как есть.
